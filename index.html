Controls:
Movement → WASD
Color Mode → "Lighting", r{Noise, Random, Flat}
Tile Size → Tile Size(Disp size: 800x800), r[25, 400]
Iterations → # of Splits on polygons, r[1, 4]
Add/Remove Colors → Add/Remove Colors, r[1, 29]
Change Colors → Press on colors on the right side


Recomendations:
More iterations lower framerate, so raise tile size
to combat that.
Darker color combinations work better with the
noise function.
Less is more color-wise.
